# My-OS

> A Personal Intelligence Operating System.

My-OS is the source of truth for my thinking, learning, career, writing, and long-term knowledge.

It is designed to help me:

- Think better.
- Learn continuously.
- Build AI products.
- Record important lessons.
- Create original work.
- Grow my career.
- Preserve how my thinking evolves.

---

# Philosophy

This repository is **not**:

- A note-taking app
- A diary
- A to-do list
- An AI agent

This repository **is**:

- A thinking system
- A knowledge graph
- A writing workspace
- A career companion
- A long-term memory
